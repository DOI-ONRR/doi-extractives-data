<<<<<<< b16d34bb08d24dddee84f80cf6103079ed4ffb33 4.1 ======= 5.1 >>>>>>> set up kss #layout.container Container

  • .container-left
    8/12 width container that floats left
  • .container-right
    4/12 width container that floats left
  • .container-half
    6/12 width container that floats left
  • .container-padded
    full width container with padding

<<<<<<< b16d34bb08d24dddee84f80cf6103079ed4ffb33 4.2 ======= 5.2 >>>>>>> set up kss #layout.slabs Slabs

Slabs are full-width blocks of color. Slabs should contain containers to center content. Slabs can can have background color styles applied. By default is a neutral background.
