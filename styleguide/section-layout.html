4 #layout Layout

Site layout styles

4.1 #layout.container Container

  • .container-outer
    should be placed on an outer div for appropriate padding
  • .container-left
    7/12 width container that floats left
  • .container-right
    5/12 width container that floats left
  • .container-left-8
    8/12 width container that floats left
  • .container-right-4
    4/12 width container that floats left
  • .container-half
    6/12 width container that floats left
  • .container-padded
    full width container with padding

4.2 #layout.slabs Slabs

Slabs are full-width blocks of color. Slabs should contain containers to center content. Slabs can can have background color styles applied. By default is a neutral background.
